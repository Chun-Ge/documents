# 系统分析与设计期末个人总结

15331172, [Yizhen Li](https://github.com/alexandrali3)

## 课程学习自我总结

本次项目我本来是负责用户调研和后台测试，后来希望在实践中学习Golang开发，因此，参与了后台开发的工作。

在后台开发的部分，我主要的负责部分有：

- 构建Xorm实体，生成并关联MySQL数据库表和关联
- 部分功能逻辑实现：登陆、登出、注册、收藏帖子、举报（帖子、回复）
- 参与第三阶段Fix API

本次体会最深的是关于JWT中间件的设计，因为Golang的JWT中间件实现目前并不适用于我们的逻辑，因此，我新增了一个JWT中间件用来作为一个Pre Handler，这样就能在每次进入route之前进行一次user authentication，将验证功能分离出来。

其次，在Xorm对数据库表的创建过程中，因为Xorm不支持外键约束，所以我需要依照数据库结构文档手动构建外键约束。在这过程中，我意识到在团队项目开发过程中，不同阶段的工作之间的相互的影响，在本阶段开发的同时，还可以对之前的阶段进行评审。
